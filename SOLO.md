# SOLO — анализ своих финансов (один пользователь)

Базовая установка — в [START.md](START.md). Здесь — как пользоваться после установки.

В сессии Claude Code:

- `/finance-report июнь-июль` — отчёт по стандартному флоу (спросит, обновлять ли).
- «покажи баланс», «сколько реально доступно?», «сколько потратил в июле по категориям»,
  «какие подписки», «сравни июнь и июль».

**Инструменты MCP:** `refresh`, `list_accounts`, `get_transactions`, `status`, `reconcile`.
Все читают только твою локальную базу. Всё локально, только чтение.

Нужен семейный режим (несколько человек, сводный отчёт)? → [FAMILY.md](FAMILY.md).
