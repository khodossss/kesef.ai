# FAMILY — семейный бюджет

Каждый член семьи ставит коннектор у себя ([START.md](START.md)) и обновляет СВОИ счета
на своей машине. Отчёт можно построить по своим счетам (`own`) или по всей семье
(`household`). Кредов между машинами не ходит — только производные данные.

## Подключение (setup-дельта поверх START)

Два режима хранения — выбери один:

### relay (рекомендуется: руками ничего)

1. Один член: в чате «создай семейный бюджет» → `family_create` вернёт **токен**.
2. Разошли токен остальным (любой мессенджер).
3. Каждый остальной: «подключись к семье по токену …» → `family_join`.
4. Опционально задай имя: `family_status` с `setInfo: "Дима"` (иначе ты «Юзер N»).

Требуется один раз развёрнутый relay (Cloudflare Worker) — см. [relay/README.md](relay/README.md).
Токен — это как пароль: он в `.env`, не пересылай его лишним людям.

### local (файлы синхронизируешь сам)

1. Общая папка (OneDrive/Dropbox/…): `family_create` с `mode: "local"`, `dir: "<папка>"`.
2. Остальные: `family_join` с тем же `dir`.
3. Синхронизацию папки обеспечиваешь ты (облако/ручной перенос).

## Анализ

Те же вопросы, что в [SOLO.md](SOLO.md), но по всей семье:

- `/finance-report … household` — сводный отчёт: секция на каждого + сумма по семье.
- «баланс по семье», «сколько семья потратила в июле» — `scope: household`.
- `family_status` — кто в семье и когда каждый обновлялся (свежесть на виду).

Сверка идёт **по каждому отдельно**, потом суммируется: погашение карты матчится с
банком того же человека, никогда между людьми.

## Синхронизация

`family_sync` запускается сам после каждого `refresh`. Вручную — «синхронизируй семью».
